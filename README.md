- VIPER是一款图形化内网渗透工具,将内网渗透过程中常用的战术及技术进行模块化及武器化.
- VIPER集成杀软绕过,内网隧道,文件管理,命令行等基础功能.
- VIPER当前已集成55个模块,覆盖初始访问/持久化/权限提升/防御绕过/凭证访问/信息收集/横向移动等大类.
- VIPER目标是帮助红队工程师提高攻击效率,简化操作,降低技术门槛.
